# Jambo Company Homebrew Tap

This repository contains the official macOS Homebrew packages for [RunEnv Desktop](https://runenv.dev/).

## Installation

To install RunEnv Desktop via Homebrew, run the following commands in your terminal:

```bash
brew tap Jambo-company/tap
brew install --cask runenv
```

## Updates

Once installed, RunEnv Desktop will automatically update itself using its built-in updater, but you can also update via Homebrew using:

```bash
brew upgrade --cask runenv
```
